# M3U Scanner - M3U 播放列表扫描工具

功能强大的 M3U/M3U8 播放列表扫描工具，带有现代化 Web 界面。

## ✨ 功能特性

### 核心功能
- **📁 M3U/M3U8 解析** - 支持本地文件和远程 URL
- **⚡ 并发扫描** - 多线程同时检测，可调节并发数（1-100）
- **📊 详细流信息** - 显示视频/音频编码、码率、分辨率、帧率等
- **🖼️ 自动截图** - 扫描到的频道自动生成预览缩略图
- **🔍 过滤搜索** - 按名称/分组过滤，支持只显示可用频道
- **💾 结果导出** - 将可用频道导出为新的 M3U 文件

### 播放功能
- **🌐 浏览器播放** - 直接在网页中播放视频流
- **🎬 外部播放器** - 支持多种播放器

### 高级功能
- **🔧 User-Agent 设置** - 预制多种 UA，支持自定义
- **⚙️ 灵活配置** - 可调节并发数、超时时间
- **🚀 快速模式** - 仅检测可用性，跳过详细信息获取

## 🚀 快速开始

### 编译

```bash
cd m3u-scanner
go build -o m3u-scanner.exe .
```

**无需 C 编译器！** 纯 Go 编译，一个 exe 文件即可运行。

### 运行

```bash
# 默认端口 8080
./m3u-scanner.exe

# 自定义端口
./m3u-scanner.exe -port 9000
```

程序启动后会自动打开浏览器访问 `http://localhost:8080`

## 📦 可选依赖

### FFmpeg/FFprobe

用于获取详细流信息和生成缩略图。**没有也能运行**，只是无法获取详细信息。

**安装方式：**

#### Windows
1. 从 https://ffmpeg.org/download.html 下载
2. 解压后把 `ffmpeg.exe` 和 `ffprobe.exe` 放到以下任一位置：
   - **同目录**（推荐）：和 `m3u-scanner.exe` 放一起
   ```
   m3u-scanner/
   ├── m3u-scanner.exe
   ├── ffmpeg.exe
   └── ffprobe.exe
   ```
   - **系统 PATH**：添加 FFmpeg 目录到环境变量

#### macOS
```bash
brew install ffmpeg
```

#### Linux
```bash
sudo apt install ffmpeg  # Debian/Ubuntu
sudo dnf install ffmpeg  # Fedora
```

## 📖 使用方法

### 1. 加载播放列表
- **打开文件** - 点击 "📁 打开文件" 选择本地 M3U 文件
- **打开 URL** - 点击 "🌐 打开URL" 输入远程播放列表地址

### 2. 扫描频道
- 点击 **"▶️ 开始扫描"** 开始检测所有频道
- 勾选 **"快速检测"** - 仅检测可用性，速度更快（不获取详细流信息）
- 点击 **"⏹️ 停止"** 可随时中止扫描

### 3. 查看结果
- **左侧列表** - 显示所有频道及状态（绿点=可用，红点=不可用）
- **右侧详情** - 点击频道查看：
  - 📺 预览缩略图
  - 📊 流状态和响应时间
  - 📦 容器格式
  - 🎥 视频流信息（编码、分辨率、帧率、码率）
  - 🔊 音频流信息（编码、采样率、声道、码率）

### 4. 播放频道
点击 **"▶️ 播放"** 按钮，选择播放方式：
- **🌐 在浏览器中播放** - 无需安装播放器
- **🎬 外部播放器** - 选择 PotPlayer、VLC、MPV 等

### 5. 导出结果
点击 **"💾 导出"** 将所有可用频道保存为新的 M3U 文件

### 6. 设置选项
点击 **"⚙️ 设置"** 可调整：
- **并发数** (1-100) - 同时扫描的频道数量
- **超时时间** (5-120秒) - 单个频道检测超时
- **User-Agent** - 选择预制 UA 或自定义

## 💡 提示

1. **首次使用** - 先用 example.m3u 测试功能
2. **大批量扫描** - 建议勾选"快速检测"，完成后再详细扫描部分频道
3. **网络问题** - 如遇超时可增加超时时间或降低并发数
4. **User-Agent** - 某些源可能需要特定 UA，尝试切换不同 UA

## 📝 常见问题

**Q: 为什么有些频道显示不可用？**
A: 可能是网络问题、源失效、需要特定 User-Agent，或地区限制。

**Q: 缩略图无法生成？**
A: 需要安装 FFmpeg 并确保 ffmpeg.exe 在程序同目录或 PATH 中。

**Q: 网页播放器无法播放？**
A: 部分流格式浏览器不支持，建议使用外部播放器。

**Q: 扫描很慢怎么办？**
A: 勾选"快速检测"或增加并发数（注意过高可能被限流）。
